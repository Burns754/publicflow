❓ TROUBLESHOOTING

"Port 8000 already in use?"
  → Change port in .env or use: PORT=8001 bash start.sh

"ModuleNotFoundError?"
  → pip install -r requirements.txt

"Need OpenAI?"
  → Optional - works without (uses rule-based)
  → Add to .env: OPENAI_API_KEY=sk-your-key

"Want Docker?"
  → docker-compose up
